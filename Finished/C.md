## Setting C Environment

1. Download MinGW：https://bit.ly/mingw10   => mingw-17.1-without-git.exe 

2. 设置GCC环境变量 C:\MinGW\bin

3. CMD 检查： g++ --version

4. VS install **Code Runner** and **c/c++**

   1. Code Runner: Ctrl + Alt + N  start
   2. Ctrl + Alt + M stop
